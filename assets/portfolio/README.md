# Portfolio Asset Set

This folder stores web-safe versions of the UAV and robotics materials provided for the portfolio update.

Original MOV and DNG files are not committed because they are too large or not suitable for direct web delivery. DNG files were converted to JPEG previews, and the MOV was converted into three frames plus a lightweight 8-second loop.
